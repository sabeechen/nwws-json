NWWS JSON API

A JSON API for retrieving information from the NWWS Open Interface

Why?

NWWS Weather reports, alerts, etc for the United States can only be accessed via a satelite connection, by scraping the NWWS website, or by subscribing to an authenticated XMPP chat room. For most puposes, this is very cumbersome. This website provides access to the same information through a JSON API for the sake of convenience. This website is being hosted on a server that also subscribes to the XMPP server publishing NWWS's information.

How?

The JSON API can be accessed in three ways:

  • /centers is a human-readable webpage listing the centers with published information. You can find the 4 digit name of a center close to you here.
  • /center/%station_name% lists all messages collected from a given station.
  • /message/%station_name%/%wmo_product_id% returns that latest message a station has published for a given WMO product id.

A message returned by the API has the following format. The fields reflect those described at the bottom of NWWS's Configuration page. 

{
    "date": "2020-05-24T00:00Z",
    "awips_id": "SYNBOU",
    "wmo_product_id": "NZUS99",
    "subject": "KBOU issues SYN valid 2020-05-24T15:47:00Z",
    "issuing_center": "KBOU",
    "id": "10948.5331",
    "message": "..."
}

Caveats

  • This website and the service it provides is not associated with NWWS, NOAA, or any other organization. Its put together and maintained by one guy who found it useful for himself and thought others might like it too.
  • Only the most recent message for each wmo product id is returned.
